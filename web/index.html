Automated Machine-Checked Hybrid System Safety Proofs

By Herman Geuvers, Dan Synek, Adam Koprowski, Eelis van der Weegen.

This page provides documentation and sources for our Coq implementation of a certified automated hybrid system safety verifier. This development is the subject of our submission (currently pending review) for ITP 2010. A draft version is available.
